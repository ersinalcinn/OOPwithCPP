-----MENU FUNCTION
--1)Start a new game:
In this case, I create a new class object. The game is played and the game remains open as long as the game is not finished according to the selected situations and appears as an active game.
Then I save the active games in the activeGame vector array and through this array I can access all the data of this game.
--2)Unfinished active games are listed, and any active game selected can be switched to and resumed.
As game features: it is easier to choose the game by specifying board size and game type.
3)All non-empty cells in all active games are counted. These cells are compared between games and the most full game is printed on the screen.
4)exit
